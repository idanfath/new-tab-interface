## Installation

```bash
# Clone the repository
git clone https://github.com/idanfath/mnt_interface.git

# Install dependencies
pnpm install

# Start development server
pnpm dev
```

## Technology Stack

- React 18
- TypeScript
- Tailwind CSS
- Radix UI Components
- Framer Motion
- OGL (WebGL)
